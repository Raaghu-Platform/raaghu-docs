# 介绍

Raaghu设计系统使用React 18构建，提供了强大的、模块化的架构，完全兼容ABP商业版。

它提供了用于生产的终端用户应用页面。可定制的仪表板模板，包括30多个应用页面、预定义的主题、模块以及端到端的文档。

## 入门

本教程指导您如何使用启动模板创建和运行新的Web应用程序。
